# Tommy Goodman

Full-stack developer based in Helensville, Auckland.

I spent 15 years as an avionics technician in the Royal New Zealand Air Force
diagnosing aircraft electrical and electronic systems, chasing intermittent
faults, and making sure nothing broke at the wrong moment. I retrained as a
developer and completed a Full-Stack Diploma at Mission Ready HQ and a
Professional Certificate in Software Engineering at Institute of Data (AUT
accredited).

Most recently I did a 10-week internship at PolicyCheck, where I integrated
Listmonk as a Docker service into their Next.js platform, built a Node.js
seeding middleware for dev environment setup, and debugged networking issues
across a multi-service Docker Compose stack.

## Projects I'm proud of

### Whenuapai Key Press Manager

Full-stack web app I built to replace the paper-based key tracking system at
RNZAF Whenuapai. Real-time dashboard, overdue alerts, full audit history,
access control. Cut the time spent on key management by 80%.

### Presence Guard

Visitor management system for a restricted RNZAF building. Replaced
handwritten visitor books with a searchable digital record — visitor details,
escort info, time in/out, daily exportable log for the duty officer.
